## How to install a model
1. En la imagen, elige el modelo que quieras descargar desde Hugging Face.
2. Copia el `repo_id` y `filename` del modelo y actualízalos en `download_models.py`.
3. Ejecuta en terminal:

```bash
python download_models.py
```

> La carpeta `models` **se crea automáticamente** si no existe.
